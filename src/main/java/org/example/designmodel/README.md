# 设计模式专题

GoF 23 种设计模式的 Java 实现与真实场景应用。

---

## 三大分类

| 类型 | 模式 | 关键词 |
|------|------|--------|
| **创建型**（5种） | 单例、工厂、抽象工厂、建造者、原型 | 对象怎么创建 |
| **结构型**（7种） | 适配器、装饰器、代理、外观、桥接、组合、享元 | 类和对象怎么组合 |
| **行为型**（11种） | 策略、模板方法、观察者、责任链、命令、迭代器等 | 对象之间怎么协作 |

---

## 学习建议

不要死记模式定义，重点理解**每个模式解决了什么问题**。

高频应用的模式（先学这些）：
- **单例**（线程安全实现方式）
- **工厂/抽象工厂**（Spring Bean 创建）
- **装饰器**（IO 流体系，BufferedInputStream 就是典型）
- **代理**（AOP 核心，详见 `proxy_aop/`）
- **观察者**（事件驱动，Spring 事件机制）
- **策略**（替代 if-else，Spring `HandlerMapping` 就是策略模式）
- **责任链**（Servlet Filter、Netty Pipeline）
