# 计算机原理专题

CPU、内存、指令集等底层硬件原理，理解程序运行的物理基础。

---

## 核心知识点

| 方向 | 重点内容 |
|------|---------|
| **CPU** | 流水线，分支预测，乱序执行，缓存一致性（MESI协议） |
| **内存** | CPU 三级缓存，缓存行，伪共享，内存屏障 |
| **指令集** | CISC vs RISC，常用汇编指令 |
| **总线 & DMA** | 内存总线，DMA 传输（零拷贝的硬件基础） |

---

## 与 Java 的关联

- CPU 缓存行 / 伪共享 → `@Contended` 注解，`Disruptor` 框架设计
- 内存屏障 → `volatile` 关键字，`happens-before` 原则
- DMA 直接内存访问 → 零拷贝的两次 DMA 传输（详见 `io/nio/` Part9）
- 乱序执行 → DCL 单例模式为什么需要 `volatile`

建议作为学习的「拔高内容」，在掌握多线程和 IO 知识后回来对照理解。
